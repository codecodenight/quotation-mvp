19 sheets, 19 categories. Key structural patterns:
- Single FOB price column (太阳能系列, 面板灯塑料)
- Multi-spec FOB columns: same product priced differently by voltage/efficacy/driver/power-level/material/warranty
- Factory cost comparison columns to the right of FOB (含税/不含税 RMB + margin formulas)
- Repeating header rows for vertical product sub-groups (工矿灯, 球泡)
- Multi-row headers with sub-columns (太阳能, 灯带)

Price variant dimensions embedded in FOB column headers: voltage (220-240V vs 185-265V), efficacy (80/90/100 lm/W), driver (Linear/IC/DOB), power-level (75%/90%), material (PP/PC), warranty (1yr/2yr/3yr).
