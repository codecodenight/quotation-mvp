# V2.8 B1 — Fill-Down Model Column Test

## Code Change

`HejiaImportMapping` now supports:

```ts
fillDownModelColumn?: boolean
```

When enabled, empty cells in the mapped model column inherit the previous non-empty model value.
This handles Excel merged-cell layouts where one model name covers several variant rows.
