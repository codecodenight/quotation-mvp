### 1. #1 德雷普灯丝灯 has high duplicate-offer count

Dry-run result:
- Valid rows: 271
- Unique products: 91
- New offers: 91
- Duplicate offers: 180

Reason:
- The source uses one model cell such as `C35` for multiple wattage/price rows.
- With the current plan, fill-down keeps rows valid, but the importer still treats all `C35` rows as the same product/factory offer.

Impact:
- If applied as-is, only one offer per model/factory is imported.
- Variant prices under the same model are skipped as duplicate offers.

Decision needed before apply:
- If this is acceptable as a family-level import, proceed.
- If each wattage must be independently quoteable, #1 needs generated model names such as `Model + Watts + Base/Size` before apply.
