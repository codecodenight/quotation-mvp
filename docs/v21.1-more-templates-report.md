# V21.1 More Quote Templates Report

## Scope

Implemented four high-frequency category quote templates using the existing V21.0 template registry pattern.

| Category | File | Sheet name | Columns | Special columns |
|---|---|---|---:|---|
| 投光灯 | `src/lib/quote-templates/floodlight.ts` | LED Floodlight | 17 | Beam Angle before FOB Price |
| 线条灯 | `src/lib/quote-templates/linear.ts` | LED Linear Light | 15 | Length (mm), no Driver, no Beam Angle |
| 球泡 | `src/lib/quote-templates/bulb.ts` | LED Bulb | 16 | Base, Shape, Luminous Efficacy |
| 灯带 | `src/lib/quote-templates/strip.ts` | LED Strips | 15 | W/m, LED Chip, LEDs/m, PCB Width |
