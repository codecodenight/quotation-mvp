## 剩余缺口为什么不应继续追

- **beam_angle 6,518 缺口**：筒灯从 12° 到 120°，球泡从 38° 到 300°，磁吸灯从 24° 到 270°。品类内无统一值，推理即编造。
- **lumens 7,685 缺口**：同型号不同功率的流明值完全不同，无法从品类或系列推断。
- **material 4,353 缺口**：同品类内铝/PC/ABS/玻璃并存，无规律可推。
- **driver_type 5,618 缺口**：隔离/非隔离取决于具体方案，无法从品类推断。
- **warranty 8,825 缺口**：商务条款，同品类不同供应商差异大。

## 后续约束

1. 不再为参数覆盖率发起新版本
2. 不再用宽松匹配、AI 猜测、或品类多数值推断来填参数
3. 如果未来有新 Excel 文件导入，可复用 V27.1 的精确匹配脚本自动提取
4. 参数缺口在 UI 中应展示为"无数据"而非隐藏，避免用户误以为数据完整
