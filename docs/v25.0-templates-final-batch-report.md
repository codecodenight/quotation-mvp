## 注意事项

任务文档写的是 `G4G9` sheet name: `LED G4/G9 Bulb`，但 ExcelJS/Excel worksheet name 不允许 `/` 字符。实际模板使用 `LED G4-G9 Bulb`，否则导出会抛错：

```text
Worksheet name LED G4/G9 Bulb cannot include any of the following characters: * ? : \ / [ ]
```
