# V19.1 — Quotes 参数筛选扩展报告

Date: 2026-06-22

## Summary

报价中心 `/quotes` 已增加更多结构化参数筛选器，和 V19.0 Chat 工具的搜索能力对齐。新增筛选器放在“更多筛选”折叠区，默认不占用主筛选区域；当任一新增筛选字段有值时自动展开。

## Filters Added

新增筛选：
- 驱动类型 `driverType`
- 显色指数 `cri`
- 功率因数 `pf`
- 光束角 `beamAngle`
- 最小光效 `minEfficacy`
- 最大光效 `maxEfficacy`

## Server-Side Logic

- `driverType`, `cri`, `pf`, `beamAngle` 使用现有的参数 OR 逻辑：匹配值，或产品缺少该参数记录。
- `minEfficacy` / `maxEfficacy` 通过 `getProductIdsByParamRange("luminous_efficacy", ...)` 查询。
- 功率范围和光效范围同时存在时取 product id 交集。
- 大 product id 列表按 400 个分块，避免 Prisma 在关联查询时 panic。

## UI Layout

现有主筛选保持两行：
- 第一行：搜索产品 / 品类 / 工厂 / 筛选按钮
- 第二行：功率范围 / IP / 色温 / 电压 / 材质 / 排序

新增第三块：
- `<details>` 折叠区标题为“更多筛选”
- 内容为 6 个字段：驱动类型、显色指数、功率因数、光束角、最小光效、最大光效
- 有值时默认展开
