# V21.0 Quote Template Engine Report

Generated: 2026-06-22

## Scope

Implemented category-specific quotation export infrastructure and the first category template for 面板灯.

The export decision is all-or-nothing:
- If every quote item has the same category and a registered template exists, export with that category template.
- If items span categories, have missing categories, or the category has no template, export with the existing generic quotation workbook.

## Panel Template Columns

| Col | Header | Source |
|---|---|---|
| A | No. | Row index |
| B | Model No. | `products.model_no` |
| C | Power | `params.watts + "W"` |
| D | Size (mm) | `params.size_display` or dimensions |
| E | Material | `params.material` or `products.material` |
| F | CCT | `params.cct + "K"` |
| G | CRI | `"Ra" + params.cri` |
| H | PF | `params.pf` |
| I | Voltage | `params.voltage + "V"` |
| J | Driver | `params.driver_type` |
| K | IP | `"IP" + params.ip` |
| L | FOB Price (USD) | Calculated sale price |
| M | MOQ (PCS) | Cleaned MOQ |
| N | CTN QTY | `supplier_offers.ctn_qty` |
| O | CTN Size (cm) | `ctn_length × ctn_width × ctn_height` |
| P | Packing Volume (m³) | `L × W × H / 1,000,000` |
