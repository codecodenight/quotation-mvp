# V19.0 — Chat 参数搜索扩展报告

Date: 2026-06-22

## Summary

Chat 工具的产品搜索能力已扩展到结构化参数层。`search_products` 现在可以按电压、色温、IP、材质、驱动类型、CRI、PF、光束角和光效范围筛选；`compare_factories` 也支持常用规格参数。

## Tool Parameters Added

### search_products

新增参数：
- `voltage`
- `cct`
- `ip`
- `material`
- `driver_type`
- `cri`
- `pf`
- `beam_angle`
- `min_efficacy`
- `max_efficacy`

### compare_factories

新增参数：
- `voltage`
- `ip`
- `driver_type`
- `material`

## Implementation Notes

- 新增范围筛选复用 `src/lib/product-filters.ts` 的 `getProductIdsByParamRange()`。
- `watts` 与 `luminous_efficacy` 范围同时存在时会取 product id 交集。
- 精确参数匹配沿用报价页的 OR 逻辑：匹配该参数值，或该产品没有这个参数记录，避免因源数据缺失把潜在产品排除掉。
- 为避免 Prisma 在“大量 id + 关联读取”场景下触发 query engine panic，产品 id 条件按 400 个一组分块。
